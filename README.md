The BookBaseV2 is the second version of the BookBase, a simple database built for study and demonstration of CRUD on JavaScript. This second version, in construction in this repository, has the addition of a HTML interface with the objective of making the use of it easier and cleaner than the first version. 

## Features

- Create, read, update and delete books and authors; 
- Edit their relationship to eachother (who wrote what book?);

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/OffworldAstronaut/BookBaseV2.git
   cd BookBaseV2
   ```

2. Install dependencies:
   ```bash
   npm i
   ```

3. Start the server:
   ```bash
   npm run dev
   ```

4. Open your browser and type in the address bar:
   ```
   http://localhost
   ```
